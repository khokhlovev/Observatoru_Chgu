Встроенные HTML-элементы

HTML определяет длинный список доступных встроенных тегов, полный список которых можно найти на Mozilla Developer Network.

  • Для выделения текста жирным шрифтом используйте <strong>.
  • Чтобы выделить текст курсивом, используйте <em>.
  • Следует использовать сокращения, подобные HTML, <abbr> с необязательным title атрибутом для полной фразы.
  • Следует использовать цитаты, такие как Марк Отто <cite>.
  • Должен использоватьсяудаленный текст <del> и должен использоваться вставленный текст <ins>.
  • Надстрочный текст использует <sup> и подстрочный текст использует <sub>.

Большинство этих элементов оформлены браузерами с небольшими изменениями с нашей стороны.
